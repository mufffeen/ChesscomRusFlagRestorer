# Chess.com Russian Flag Restorer

**Chess.com Russian Flag Restorer** — это расширение для Chrome, которое заменяет нейтральный флаг на сайте Chess.com на российский.  

⚠️ **Внимание:** флаги Беларуси будут также отображаться как российские.

## Установка
1. Скачайте ZIP-файл расширения.
2. Откройте Chrome и перейдите на страницу расширений: `chrome://extensions/`
3. Включите режим разработчика (переключатель в правом верхнем углу).
4. Нажмите **"Загрузить распакованное расширение"**.
5. Выберите папку с **распакованным** содержимым ZIP-файла.
6. Расширение готово к использованию — обновите страницу Chess.com, и флаги изменятся.

> ⚠️ Важно: это расширение работает только на настольной версии Chrome и меняет только визуальное отображение флагов.
